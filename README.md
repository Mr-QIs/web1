# 豪帅网络服务 - 企业网站首页

## 项目简介

这是一个现代化的企业网站首页，专为"豪帅网络服务"公司设计。网站采用 HTML5 + CSS3 + JavaScript 技术栈，实现了丰富的动态科技感视觉效果和响应式布局。

## 主要特性

### 🎨 动态科技感效果
- **粒子动画背景**：支持鼠标交互的动态粒子效果
- **渐变动画**：背景渐变色随时间动态变化
- **光线特效**：导航栏和按钮的发光效果
- **平滑滚动动画**：元素进入视窗时的渐入动画
- **鼠标悬停效果**：卡片和按钮的交互反馈

### 📱 响应式设计
- 适配桌面、平板和移动设备
- 移动端汉堡菜单
- 弹性布局和网格系统
- 触摸友好的交互设计

### 🎯 页面结构
1. **顶部导航栏**：固定导航，支持平滑滚动
2. **Hero/Banner区域**：动态背景 + 企业品牌展示
3. **核心展示区**："豪帅网络服务"重点展示
4. **服务介绍区**：四大核心业务展示
5. **关于我们区**：公司介绍和技术栈
6. **联系区**：联系方式和联系表单
7. **底部Footer**：包含ICP备案号和公安备案号

### ⚡ 技术实现
- **HTML5**：语义化标签，现代Web标准
- **CSS3**：Flexbox/Grid布局，动画和过渡效果
- **JavaScript ES6+**：模块化代码，现代JavaScript特性
- **Canvas API**：粒子动画系统
- **Intersection Observer API**：性能优化的滚动动画
- **CSS Grid & Flexbox**：现代布局技术

## 使用方法

### 1. 直接访问
直接用浏览器打开 `index.html` 文件即可预览网站效果。

### 2. 本地服务器（推荐）
```bash
# 使用Python启动本地服务器
python -m http.server 8000

# 或使用Node.js
npx serve .

# 然后在浏览器访问 http://localhost:8000
```

### 3. 在线部署
将所有文件上传到Web服务器即可。

## 自定义配置

### 修改公司信息
在 `index.html` 中搜索并替换以下内容：
- 公司名称：搜索 "豪帅网络服务"
- 联系方式：修改底部联系信息
- 服务内容：更新服务卡片内容

### 修改颜色主题
在 `styles.css` 中修改CSS变量：
```css
/* 主要颜色变量 */
--primary-color: #00ffff;    /* 青色主题 */
--secondary-color: #ff00ff;  /* 紫色强调色 */
--bg-dark: #0a0a0a;         /* 深色背景 */
--bg-secondary: #1a1a2e;    /* 次要背景 */
```

### 调整动画效果
在 `script.js` 中修改参数：
- 粒子数量：修改 `particles.push(new Particle())` 循环次数
- 动画速度：调整 `speed` 变量值
- 动画延迟：修改各元素的 `transition-delay` 值
